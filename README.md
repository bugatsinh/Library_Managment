# Library Management System (Python)

A simple Library Management System built using Python and JSON for data storage.

## Features

* Add new books to the library.
* View all available books.
* Borrow books (changes status from available to unavailable).
* Return borrowed books (changes status back to available).
* Search books by name.
* Stores all book records in a `books.json` file.

## How It Works

The program uses a menu-driven interface where users can choose different options such as adding, viewing, borrowing, returning, or searching for books. Book information is stored as dictionaries inside a JSON file, making the data persistent even after the program is closed.

## Technologies Used

* Python
* JSON File Handling

## Learning Concepts

This project demonstrates:

* Functions
* Lists and Dictionaries
* File Handling
* JSON Data Storage
* Loops and Conditional Statements
* Basic CRUD Operations (Create, Read, Update, Delete)
